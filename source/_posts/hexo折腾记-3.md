# 1- 首页博客展示调整  文章展示部分 点击阅读全文查看文章所有内容

第一步：你需要在你的md文件中加上
```
title: hexo折腾记-3
date: 2017-08-30 13:23:33
tags:
- hexo
description: hexo折腾记-3
```

最后这个description这部分

第二步在你的md文件中加上more注释,如本猿这篇文档，我就希望有个标题，剩下内容都在阅读全文中查看。
```
<!-- more -->
```
